# music-dl

A script that simplifies downloading and tagging music from youtube.

## Dependencies

- [youtube-dl](https://youtube-dl.org/)
- ffmpeg

## Installation

`git clone https://github.com/segf00lt/music-dl && cd music-dl/ && sudo make install ;`
