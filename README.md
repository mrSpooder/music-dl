# music-dl

A script that simplifies downloading music from youtube and embedding it with the correct metadata.

## Dependencies

- [youtube-dl](https://youtube-dl.org/)
- [beets](https://beets.io/)

## Installation

`git clone https://github.com/mrSpooder/music-dl ;
cd music-dl/ ;
sudo make install ;`
